{# Shared PWA install widget. Chrome/Edge/Android fire `beforeinstallprompt`; we capture it and show a real "Install app" button (the native omnibox icon is easy to miss). iOS Safari never fires that event and shows no native prompt, so we detect iOS and surface a manual "Add to Home Screen" hint instead. #}
